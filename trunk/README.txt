Ajouter un formulaire à votre panier de votre boutique WooCommerce pour que vos clients puissent faire un don

== Description ==

Le plugin WooCommerce Add Donation vous permet de proposer aux vos clients de votre boutique WooCommerce la possibilité de faire un don.

Langues disponibles / Languages :

* Français
* English

Fichier woocommerce-add-donation.pot disponible / woocommerce-add-donation.pot file available



== Installation ==
1. Ajouter le répertoire 'woocommerce-add-donation' dans le répertoire '/wp-content/plugins/' / Upload the full directory into your '/wp-content/plugins' directory
2. Activer le plugin via le menu 'Plugins' dans WordPress / Activate the plugin at the plugin administration page
3. Aller dans 'Réglages->WooCommerce Add Donation' dans l'interface d'administration pour sélectionner vos options / Open the plugin configuration page, which is located under 'Option->Réglages->WooCommerce Add Donation'
